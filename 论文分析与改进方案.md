# 论文分析与改进方案：基于去噪异构图方法改进 MMFSynergy 宏观编码器

## 1. 问题背景

### 1.1 现状

MMFSynergy 的宏观编码器基于异构图注意力网络（HAN），在 drug-protein-sideeffect 异构网络上进行链接预测以学习全局节点嵌入。

### 1.2 数据质量评估结果

通过扰动实验（`assess_macro_data_quality.py`）发现异构图数据存在噪声问题：

**边扰动实验**（节点不变，随机去除边）：

| 去除比例 | AUC | AP |
|---------|------|------|
| 0%（完整图） | 0.8784 | 0.8303 |
| 5% | 0.8843 (+0.7%) | 0.8413 (+1.3%) |
| 10% | 0.8765 (-0.2%) | 0.8297 (-0.1%) |
| **20%** | **0.8860 (+0.9%)** | **0.8440 (+1.6%)** |
| 30% | 0.8722 (-0.7%) | 0.8175 (-1.5%) |

**节点扰动实验**（边不变，随机去除节点）：

| 去除比例 | AUC | AP |
|---------|------|------|
| 0%（完整图） | 0.8784 | 0.8303 |
| 2% | 0.8801 (+0.2%) | 0.8322 (+0.2%) |
| 5% | 0.8780 (-0.05%) | 0.8291 (-0.1%) |
| 10% | 0.8724 (-0.7%) | 0.8133 (-2.0%) |

### 1.3 问题诊断

**如果数据质量好**，随机去除边/节点应导致性能单调下降——因为真实信息被删除了。

**但实际表现是**：去除 5% 或 20% 的边后性能反而提升，说明原始图中存在**噪声边（false positive edges）**。这些虚假的交互关系干扰了 GNN 的消息传递机制，去掉它们反而让模型学到更准确的拓扑模式。

## 2. 四篇论文概述

### 论文1：HGNN-WI

- **标题**：Heterogeneous Graph Neural Networks with Weak Information
- **发表**：Expert Systems With Applications
- **核心问题**：异构图中信息缺失（结构不完整、特征缺失、标签稀疏）
- **方法**：
  - 基于 metapath 的子图特征补全（PageRank 扩散）
  - kNN 增强图的特征互补
  - 多视角对比学习（原始图 vs 增强图 vs 打乱图）
- **适用场景**：数据不完整、缺失严重的异构图

### 论文2：NoiseHGNN

- **标题**：Synthesized Similarity Graph-Based Neural Network For Noised Heterogeneous Graph Representation Learning
- **发表**：AAAI 2025
- **核心问题**：异构图中存在噪声边（错误链接）
- **方法**：
  - 合成相似性图（Similarity Graph Synthesizer）：基于节点特征余弦相似度 + kNN 稀疏化
  - 相似性感知注意力（Similarity-aware Attention）：用相似性图调整 GNN 注意力权重，降低噪声边权重
  - Metapath-Target 对比学习：对比 metapath 图（来自噪声图）和 target 图（来自相似性图）
- **适用场景**：异构图中有错误链接/噪声边

### 论文3：HGNN-AR2

- **标题**：Heterogeneous Graph Neural Network with Adaptive Relation Reconstruction
- **发表**：Neural Networks 2025
- **核心问题**：metapath 构建的邻接关系中存在类间错误连接和类内连接缺失
- **方法**：
  - 图自编码器（GAE）预测边存在概率
  - 插值融合：P = α·Ã + (1-α)·A（重建邻接与原始邻接的加权组合）
  - Gumbel-Softmax Bernoulli 采样稀疏化
  - GCN 分类
- **适用场景**：metapath 连接质量差、类间噪声连接多

### 论文4：SAGNN

- **标题**：Multi-level Feature Subgraph Aggregation Graph Neural Network
- **发表**：Information Fusion
- **核心问题**：忽略同构/异构邻居的交互差异，缺乏多层级特征提取
- **方法**：
  - 将异构图分解为同构子图和异构子图
  - 分别用不同的聚合模块处理
  - 加权系数融合同构/异构表示
- **适用场景**：需要区分同构/异构邻居的多层级特征挖掘

## 3. 适用性分析

### 3.1 与 MMFSynergy 问题的匹配度

| 论文 | 匹配度 | 理由 |
|------|--------|------|
| 论文2 (NoiseHGNN) | ★★★★★ | **直接解决噪声边问题**，与我们的数据质量问题完全对应 |
| 论文3 (HGNN-AR2) | ★★★★☆ | 自适应重建关系可过滤噪声，但原设计用于 metapath 场景 |
| 论文1 (HGNN-WI) | ★★★☆☆ | 侧重信息缺失而非信息噪声，方向略有偏差 |
| 论文4 (SAGNN) | ★★☆☆☆ | 侧重特征提取架构，不直接处理噪声问题 |

### 3.2 推荐方案：采用论文2（NoiseHGNN）的相似性感知去噪思路

#### 为什么 NoiseHGNN 最适合？

**1. 问题完全对应**

NoiseHGNN 论文开篇就指出：*"Real-world graph data environments intrinsically exist noise (e.g., link and structure errors) that inevitably disturb the effectiveness of graph representation."* 这与我们在扰动实验中观察到的现象完全一致——MMFSynergy 的宏观异构图中存在噪声边，去除部分边后性能反而提升。

**2. 核心机制可直接迁移**

NoiseHGNN 的相似性感知注意力机制可以集成到 MMFSynergy 现有的 HAN 编码器中：

- 当前 MMFSynergy 的 `HANLayer` 使用标准 GATConv，对所有边一视同仁
- 引入相似性图后，可以**给噪声边分配更低的注意力权重**，从而抑制噪声传播
- 相似性图基于节点特征独立构建，不依赖原始图的拓扑结构，提供了独立的校验信号

**3. 对比学习增强鲁棒性**

Metapath-Target 对比学习可以进一步：
- 从噪声图中提取有效的语义关系（metapath 图）
- 从相似性图中提取可靠的连接模式（target 图）
- 通过对比两者来学习更鲁棒的节点表示

**4. 实验验证充分**

NoiseHGNN 在 30% 噪声率的 5 个数据集上均取得 SOTA，在 DBLP 和 IMDB 上比同期方法提升 3-6%。我们的数据在去除约 20% 边后性能最优，暗示噪声率约为 15-25%，在 NoiseHGNN 的有效范围内。

#### 论文3（HGNN-AR2）作为备选/补充

HGNN-AR2 的自适应关系重建也有价值：
- GAE 边预测器可以评估每条边的置信度
- Bernoulli 采样可以去除低置信度边
- 但其设计基于 metapath 邻接矩阵，需要较多适配

## 4. 具体改进方案

### 4.1 方案概述

在 MMFSynergy 的 `MacroEncoder` 中引入**相似性感知去噪模块**，核心思路：

```
原始异构图 G
     │
     ├──→ 节点特征投影 → 余弦相似度 → kNN 稀疏化 → 相似性图 Gθ
     │
     ├──→ HANLayer (原始图) ──→ H_original
     │
     └──→ HANLayer (相似性图) ──→ H_similarity
                                      │
                对比学习损失 ←──────────┘
                                      │
                融合表示 → 链接预测损失 ←┘
```

### 4.2 关键改动

**Step 1：构建相似性图**

```python
# 对每种节点类型，基于特征计算余弦相似度
Z = W_type @ X_type  # 投影到统一空间
S = cosine_similarity(Z, Z)  # 相似度矩阵
A_sim = knn_sparsify(S, k=15)  # kNN 稀疏化
```

**Step 2：相似性感知注意力**

在 GATConv 中，用相似性图的权重调节注意力系数：
- 如果两个节点在相似性图中有连接，保持/增大注意力
- 如果没有连接，降低注意力（暗示该边可能是噪声）

**Step 3：对比学习正则化**

添加对比损失项：
```
L_total = L_link_pred + λ * L_contrastive
```

### 4.3 预期效果

1. 噪声边被自动降权 → 宏观嵌入质量提升
2. 扰动鲁棒性改善 → 去除边后性能单调下降（符合预期）
3. 下游协同预测精度可能因更好的宏观嵌入而提升

## 5. 实验计划

### 5.1 验证实验：对比去噪前后的扰动鲁棒性

1. **Baseline**：当前 MacroEncoder（标准 HAN）
2. **改进版**：加入相似性感知去噪的 MacroEncoder
3. **评估**：在相同扰动率 (0%, 5%, 10%, 20%, 30%) 下对比 AUC/AP

**预期结果**：
- 改进版在 0% 扰动（完整图）时 AUC/AP 应高于 Baseline
- 改进版应呈现更平滑的性能下降曲线

## 6. 结论

综合四篇论文的分析，**论文2（NoiseHGNN）**的相似性感知去噪方法与 MMFSynergy 的数据质量问题最为匹配。该方法：

1. 直接针对异构图中的噪声边问题
2. 核心技术（相似性图 + 感知注意力 + 对比学习）可集成到现有 HAN 架构
3. 已在多个数据集上验证有效性，噪声率范围覆盖我们的场景
4. 实现复杂度适中，主要修改集中在 `MacroEncoder` 模块

建议优先实现相似性图构建和相似性感知注意力两个核心模块，然后通过扰动鲁棒性实验验证改进效果。
